## 💡 設計理念

| 原則 | 說明 |
|---|---|
| **做中學** | 每個階段以可運行的 Project 為終點，而非堆砌知識點 |
| **雙體系互補** | 等級制度回答「我是誰」，路線圖回答「學什麼」 |
| **硬性通過標準** | 每個等級有不可妥協的硬指標，過了才算數 |
| **梯隊互助** | 高等級帶低等級，構建學習生態 |
| **多路線支援** | 非技術 / 技術 / 雙刀流三條升級路徑，各取所需 |
